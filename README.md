# 📊 Social Media Analysis using Machine Learning

A comprehensive Machine Learning project that analyzes different aspects of social media data using supervised and unsupervised learning techniques. This project includes sentiment analysis, engagement prediction, and influencer segmentation using real-world social media datasets.

---

## 🚀 Project Overview

This project consists of three independent Machine Learning tasks:

### 1️⃣ Tweet Sentiment Analysis

* **Dataset:** Twitter posts dataset (`all_tweets.csv`)
* **Algorithm:** Multinomial Naive Bayes
* **Objective:** Classify tweets into positive and negative sentiments.
* **Techniques Used:**

  * Text preprocessing
  * TF-IDF Vectorization
  * Train-Test Split
  * Cross Validation
  * Sentiment Classification

---

### 2️⃣ Social Media Engagement Prediction

* **Dataset:** Social Media Engagement Dataset (`social_media_engagement_dataset.csv`)
* **Algorithm:** K-Nearest Neighbors (KNN)
* **Objective:** Predict engagement levels based on post characteristics.
* **Features Include:**

  * Likes
  * Comments
  * Shares
  * Views
  * Follower Count
  * Content Type
  * Platform
  * Sentiment

---

### 3️⃣ YouTube Influencer Segmentation

* **Dataset:** YouTube Influencer Dataset (`social media influencers - youtube.csv`)
* **Algorithm:** K-Means Clustering
* **Objective:** Group influencers into meaningful clusters based on audience and engagement metrics.
* **Techniques Used:**

  * Data Cleaning
  * Feature Scaling
  * PCA Visualization
  * Silhouette Score Analysis
  * Cluster Interpretation
